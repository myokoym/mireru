## Usage

### Launch

    $ mireru [OPTION]... [FILE]...

If no argument, then search current directory.

### Options

-R, --recursive<br />
    recursive search as "**/*"

-f, --font NAME<br />
    set font such as "Monospace 16"

### Keybind

#### Common

n: next<br />
p: prev<br />
r: reload<br />
e: expand path<br />
q: quit<br />

#### Scroll

h: left<br />
j: down<br />
k: up<br />
l: right<br />

#### Scale

+: larger<br />
-: smaller<br />

#### Special

T: thumbnail

#### Image

f: fits window size<br />
o: original size<br />

#### Text

f: change font (at random)<br />

#### Video

space: play/pause<br />
