Usage

Launch

    $ mireru [OPTION]... [FILE]...
If no argument, then search current directory.

Options

-h, --help
    show help message
-f, --font NAME
    set a font such as “Monospace 16”

Keybind

Common

  • n: next
  • p: prev
  • e: expand/collpse
  • r: reload
  • q: quit

Scroll

  • h: left
  • j: down
  • k: up
  • l: right

Scale

  • +: larger
  • -: smaller

Image

  • f: fit window size
  • o: scale to the original size

Text

  • f: change font (at random)

Video

  • space: play/pause

PDF

  • j: next page
  • k: prev page
